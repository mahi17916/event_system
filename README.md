# Event Registration System

A Django-based web application for managing event registrations. Users can view events, register, and admins can manage events and participants through the admin panel.

---

## Features

- Browse upcoming events
- Register for events
- Admin panel to add, update, and delete events
- Optional API endpoints for future integration

---

## Tech Stack

- *Backend:* Django, Python   
- *Database:* SQLite (default)
